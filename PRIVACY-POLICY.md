# Privacy Policy – ​​TMX Universal Track Downloader

**Date: November 15, 2025**

This extension does **not collect, store, or share any user data**. Everything runs locally in the browser: intercept searches, download tracks, create a ZIP file – that's it.

**Permissions**:

- `downloads`: For file storage.

- `scripting`: UI injection on TMX pages.

- Hosts: Only TM Exchange domains for API access.

No third-party services, tracking, or analytics. Updates? Check here.

By using this extension, you agree to these terms.
